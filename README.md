# EnterpriseEasily_API

ASP.NET Core Web API (.NET 8) for the EnterpriseEasily Guitar Tab Search Platform.

## Overview

Secure REST API with Auth0 JWT authentication, MusicBrainz integration for song search, and CRUD operations for user-submitted guitar tabs.

## Build & Run

```bash
dotnet build
dotnet run --project EnterpriseEasily.API
```

Swagger UI available at `https://localhost:{port}/swagger` when running in Development mode.
